## 2. Options stratégiques formulées

### OPTION 0 : Continuité stricte Phase 4 (S26 normale)

**Description** :
Poursuivre Phase 4 S26 sans modification du cadre nutritionnel, des activités standards, ou des objectifs de phase. Appliquer un monitoring rapproché de la lombalgie (observation S26 première semaine) avant décision passage stabilisation.

**Logique justifiante** :
- Sommeil objectif atteint (7h11 > 7h10 cible S25)
- Poids très proche cible (75.6 vs 75–75.5)
- Énergie améliorée (vs S24 moyenne-basse)
- Stress contexte confirmé favorable
- Escalade lombaire modérée (2/4) et isolée 2 jours, pas **sévère** (4/4) ou persistante
- 2/3 activités complétées (conformité 67%, acceptable)

**Prescriptions S26** :
- Féculents Phase 4 strict (identique S25 : 80–120 g/j)
- Activités : réintroduction RUN-40-TEMPO-FRACTIONNE-V1 (si lombaire améliore S26 début)
- Hygiène sommeil : maintien cible (7h+ minimum, 7h30 idéal)
- Monitoring lombaire rapproché (journal douleur quotidien, progression/régression)
- Posture bureau : maintien correction S25 (alternance 45–60 min)

**Conditions d'activation** :
- Utilisateur accepte escalade lombaire modérée comme risque transitoire de Phase 4
- Lombaire résout rapidement S26 début (2–3 jours), sans escalade supplémentaire
- Poids maintient tendance modérée S26 (~75–75.2 kg fin semaine)
- Sommeil maintient > 7h00 (baseline acceptable, pas rechute S24)

**Implications si activée** :
- ✓ Poursuite momentum Phase 4, atteinte objectif fin-phase (72 kg repère)
- ✓ Durée phase 4 préservée (S26 dernière semaine) → transition stabilisation S27 programmée
- ⚠ Escalade lombaire présente → risque continue S26 si cofacteur surdosage involontaire non-élucidé
- ⚠ Déficit activité (2/3) S25 → risque répétition S26 si lombaire persiste

**Critères de blocage S26** :
- Lombaire escalade > 2/4 ou persiste + 3 jours consécutifs
- Sommeil rechute < 7h00 (signal stress/fatigue nouveau)
- Poids dérégulation (gain > 0.5 kg jour ou > 1 kg semaine)

### OPTION 1 : Modulation Progressive Phase 4 (S26 allégée)

**Description** :
Poursuivre Phase 4 objectifs poids/nutrition S26, mais réduire charge musculaire activités temporairement (1 semaine) pour investiguer et résoudre escalade lombaire. Réintroduction activités S27 si signal résout.

**Logique justifiante** :
- Escalade lombaire décohésionnée stimulus → possible cofacteur surdosage involontaire (pont+durée EF)
- Réduction 1 semaine charge permet identification si cumul variations = causal
- Nutrition déficit Phase 4 maintenu → poids continue trajectoire modérée
- Donne temps diagnostic sans rupture phase
- Moins agressif vs Option 0 (reconnaissance enjeu lombaire), moins radical vs Option 2

**Prescriptions S26** :
- Féculents Phase 4 strict : **maintient** (poids gère nutrition, pas activité)
- Activités réduction :
  - FB-30-BASE-V1 : **pause 1 semaine** (pont fessier cofacteur suspect)
  - RUN-40-EF-V1 : remplacement léger (ex. marche 20–30 min modérée, RPE 2–3, génu/lombaire tolérance test)
  - RUN-40-TEMPO : **pause 1 semaine**
- Hygiène sommeil : maintien (7h+ minimum)
- Monitoring lombaire : intensif (journal quotidien pain, position sommeil, posture bureau)
- Investigation : position sommeil, posture bureau profondeur, microtrauma quotidien

**Conditions d'activation** :
- Utilisateur accepte pause temporaire musculation (1 semaine)
- Investigation cause lombaire devient prioritaire (vs momentum Phase 4)
- Confiance cofacteur surdosage comme hypothèse driving (plutôt qu'étiologie externe)

**Implications si activée** :
- ✓ Donnée 1 semaine diagnostic (cumul variations = causal ?)
- ✓ Poids continue modéré deficit Phase 4 via nutrition (pas disruption)
- ✓ Reconnaissance escalade lombaire comme priorité santé (vs "tolérer et continuer")
- ⚠ Durée Phase 4 potentiellement prolongée (S26 modulation, pas "normal", S27 continuation ?)
- ⚠ Momentum Phase 4 ralenti (activités réduites)
- ⚠ Si cofacteur pas identified → risque déficit prolongé (dégradation motivation)

**Critères réactivation activités S27** :
- Lombaire résout complètement S26 (0/4 tous jours)
- Marche test S26 tolérance sans douleur (RPE 2–3 OK)
- Données investigation (sommeil, posture) identifie contributeur

### OPTION 2 : Clôture précoce Phase 4 / Transition Stabilisation (S26 → S27 direct)

**Description** :
Terminer Phase 4 fin S25 (immédiatement, cette semaine), transitionner vers phase stabilisation S26. Justification : escalade lombaire violate contrainte non-négociable ("priorité santé"), déficit activités (2/3), objectif poids quasi-atteint (75.6 vs 75–75.5), énergie/sommeil cible atteinte. Stabilisation S26–S27 permet reset, investigation, structuration avant itération ultérieure.

**Logique justifiante** :
- Escalade lombaire inattendue malgré contexte favorable → signal prudence Phase 4 progression
- Contrainte non-négociable "priorité santé" vs escalade modérée = tension majeure
- Poids quasi-cible (75.6 vs 75–75.5) → objectif principal Phase 4 substantially atteint
- Énergie/sommeil objectifs atteints (permis exit phase)
- Déficit activités (2/3) S25 + possible récidive S26 = adhérence risque
- Stabilisation 2 semaines permits : reset signal musculaire, investigation etiologie, consolidation hygiène sommeil/nutrition, plannification itération ultérieure

**Prescriptions S26–S27 Stabilisation** :
- Féculents phase stabilisation : ~120–150 g/j (augmentation modérée vs Phase 4 80–120)
- Activités stabilisation : modérées, low intensity (RUN-40-EF-V1 seule, FB-30-MOBI-RENFO plutôt que FB-30-BASE, VTT optionnel)
- Hygiène sommeil : maintien (7h+ baseline)
- Poids observation : stabilité ±1 kg band (pas déficit, pas gain marqué)
- Investigation lombaire approfondie : position sommeil, mobilité hanche spécifique, posture bureau ergonomie

**Conditions d'activation** :
- Utilisateur accepte prolonger stabilisation (vs raccourcir Phase 4)
- Prioriser santé lombaire et investigation vs momentum Phase 4
- 2 semaines stabilisation acceptable (délai itération ultérieure +2 semaines)

**Implications si activée** :
- ✓ Reconnaissance priorité santé (escalade lombaire pause phase aggressive)
- ✓ Reset signal musculaire (2 semaines charge réduite)
- ✓ Investigation etiologie sans pression phase objectives
- ✓ Consolidation hygiène (sommeil, nutrition, posture)
- ✗ Momentum Phase 4 disrupted (clôture anticipée 1 semaine avant plan)
- ✗ Objectif repère 72 kg pushed back (itération ultérieure, décalée +2 semaines)
- ✗ Durée total plan prolongée (Phase 4 + Stabilisation + Itération = +2–3 semaines)

**Critères sortie stabilisation (S27 fin)** :
- Lombaire complètement résolu (0/4 7 jours)
- Etiologie identified et managé (posture, position sommeil, autre)
- Sommeil maintained > 7h (hygiène consolidée)
- Poids stable ±1 kg
- Prêt itération ultérieure Phase 5 (nouvelle diète ciblée, vers 72 kg)

### OPTION 3 : Pause Phase 4 / Investigation approfondie (S26 Transition-Investigation)

**Description** :
Mettre en pause Phase 4 S26, transitionner vers phase investigation temporaire ("mini-phase" 1 semaine, non-standardisée). Nutrition augmentée légèrement (vs Phase 4), activités fortement réduites, hygiène/monitoring intensif. Objectif : investiguer étiologie lombaire en profondeur (position sommeil filmée, posture bureau ergonomie complète, microtrauma quotidien journalisé) avant décision suite S27 (reboot Phase 4 vs stabilisation).

**Logique justifiante** :
- Escalade lombaire **décohésionnée stimulus explicite** → investigation approfondie justifiée
- Aucune cause claire (pas lien direct séance, absence stimulus samedi-dimanche) → besoin données qualitatives approfondies
- Nutrition déficit Phase 4 peut être temporairement stop (1 semaine pause, puis réactivation)
- Approche précautionnaire (vs "tolérer et continuer" Option 0 ou "modifier temporaire" Option 1)
- Permet décision S27 data-driven vs prise de position aveugle

**Prescriptions S26** :
- Féculents : augmentation moderate (vs Phase 4 80–120, → 100–130 g/j "baseline neutre")
- Activités : suspension exercice renforcement (FB pause), marche légère seule (20 min RPE 2–3)
- Investigation : 
  - Position sommeil : vidéo (utilisateur ou partenaire) nuit 3–4 jours
  - Posture bureau : audit ergonomie (chaise, écran, clavier, hauteur, repose-pieds)
  - Journal douleur lombaire : journalier + trigger investigation (timing, contexte, posture antes)
  - Mobilité hanche : test ROM spécifique quotidien (progression/régression)
- Hygiène sommeil : renforcé (7h30+ target)

**Conditions d'activation** :
- Utilisateur accepte semaine investigation approfondie (vs continuer phase)
- Données qualitatives détaillées possible (vidéo, journal, audit)
- Flexibilité S27 (décision suite basée sur investigation résultats)

**Implications si activée** :
- ✓ Investigation etiologia complète (données qualitatives riches)
- ✓ Reconnaissance escalade comme symptôme nécessitant enquête (vs acceptance)
- ✓ Donnée robuste pour S27 décision
- ⚠ Phase 4 disrupted (1 semaine investigation non-productive pour poids/phase objectifs)
- ⚠ Complexité investigation (vidéo, ergonomie audit, journal) may deter use
- ⚠ Poids potentiellement gain S26 si nutrition revert vers baseline (signaux confusion S27)

**Critères décision S27** :
- Si etiologie identified (ex. position sommeil, ergonomie): poursuite Phase 4 S27 avec correction
- Si etiologie non-identified: transition stabilisation S27–S28 (reset approach)

## 4. Hypothèses structurantes et zones d'incertitude

### Hypothèses utilisées dans options
1. **Escalade lombaire cofacteur = cumul surdosage involontaire** (pont + durée EF)
   - **Évidence** : timing post-séance 3–5 jours, cumul variations (pont nouvelle, durée +50%)
   - **Incertitude** : MODÉRÉE (timing compatible, mais pas diagnostic certain)
   - **Impact options** : Option 0 accepte risque ; Option 1 teste ; Option 2/3 pauses pour investiguer

2. **Escalade lombaire étiologie = facteur postural/quotidien non-sport** (position sommeil, posture bureau)
   - **Évidence** : décohésion stimulus (escalade samedi sans nouvelle séance)
   - **Incertitude** : ÉLEVÉE (aucune donnée investigation bilan)
   - **Impact options** : Option 1/2/3 investigations privilégient ce angle

3. **Poids objectif Phase 4 "substantially atteint" à 75.6 kg** (vs cible 75–75.5)
   - **Évidence** : très proche cible, 3ème semaine phase, momentum poids OK
   - **Incertitude** : FAIBLE (données claires)
   - **Impact options** : Options 0/1 continue vers 72 kg repère ; Options 2/3 pause vers stabilisation

4. **Sommeil S25 > 7h10 cible atteint, mais < 7h20 critère S26** (borderline)
   - **Évidence** : 7h11 min observé, plan S26 mentionne "7h20"
   - **Incertitude** : MODÉRÉE (critère S26 précis ?)
   - **Impact options** : Toutes options maintiennent hygiène sommeil, critère pas driver décision

5. **Stress professionnel S25 "down comme prévu"** → favorable context
   - **Évidence** : confirmé utilisateur, correlated avec amélioration sommeil
   - **Incertitude** : FAIBLE
   - **Impact options** : Toutes options supposent context favorable continue

### Zones d'incertitude non-résolues
- ⚠ **Etiologie lombalgie samedi-dimanche** : pas diagnostic unique clear
- ⚠ **Contexte coucher tard Ven 31** : cause absent bilan
- ⚠ **Intensité réelle pont fessier vs FB-30-BASE-V1 standard** : spec unclear

## 5. Points de décision explicitement humains

### Décision 1 : Acceptabilité escalade lombaire modérée
**Question** : La douleur modérée (2/4) samedi-dimanche S25 est-elle acceptable comme risque transitoire Phase 4 (Option 0) ou priorité arrêt (Options 2/3) ?

**Nuances** :
- Si escalade **sévère** (4/4) ou **persistante** (7 jours) → Options 2/3 rationnelles
- Si escalade **modérée** (2/4) et **isolée** (2 jours) → Option 0 defensible
- **Seuil personnel** : utilisateur expertise santé lombaire, douleur tolerance

---

### Décision 2 : Timing investigation vs momentum
**Question** : Investiguer cause lombaire intensivement S26 (Options 1/3) vs accepter risque et continuer momentum Phase 4 (Option 0) ?

**Trade-off** :
- **Option 0** : momentum poids, durée plan raccourcie, mais risque escalade S26
- **Options 1/3** : investiguer, mais déficit activité, durée plan prolongée

---

### Décision 3 : Éligibilité transition stabilisation S26
**Question** : Si escalade lombaire + déficit activités demeurent S26 → transition stabilisation acceptable (vs relancer Phase 4 S27) ?

**Contexte** :
- Poids cible Phase 4 (75–75.5 kg) quasi-atteint (75.6)
- Objectif repère 72 kg (long-term, après stabilisation itérations)
- Stabilisation 2 semaines permet reset, investigation
