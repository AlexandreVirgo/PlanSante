## 2. Tensions structurantes

### Tension 1 : Escalade lombaire vs contexte favorable (MAJEURE)

**Description** :
Douleur lombaire modérée (2/4) persiste samedi-dimanche malgré :
- Stress professionnel down (contexte externe favorable)
- Sommeil amélioré (7h11 min vs 7h06 S24)
- Énergie rapportée bonne (vs moyenne-basse S24)
- Corrections ergonomiques appliquées (posture bureau, mobilité hanches)
- Absence stimulus activité clair samedi-dimanche

**Manifestation** :
```
[Contexte S25 favorable]
  ├─ Stress ↓
  ├─ Sommeil ↑ (7h11)
  ├─ Énergie ↑
  ├─ Corrections appliquées
  └─ Résultat attendu : stabilisation lombaire
        └─ Réalité observée : douleur 2/4 sam-dim (escalade vs S24 léger)
                └─ TENSION LOGIQUE
```

**Implications** :
- Hypothèse 1 : Cofacteur surdosage involontaire (cumul pont fessier nouveau + RUN-40-EF +20 min)
- Hypothèse 2 : Étiologie non-activité (position sommeil, microtrauma bureau, tension posturale quotidienne)
- Hypothèse 3 : Timing delayed (séance lun-mer, signal émerge sam-dim = décalage de 3–5 jours)

**Cadre d'analyse** : Tension entre contrainte non-négociable ("priorité santé — santé lombaire") et observation empirique (escalade). Requiert clarification étiologique pour décision S26.

### Tension 2 : Critères décision S26 ambigus (MODÉRÉE)

**Description** :
Plan S25 § "Transition vers S26" définit critères décision S26 :
```
Si sommeil > 7h20 + énergie améliorée + signaux stables 
  → continue Phase 4 S26 (dernière semaine)
Sinon si sommeil < 7h00 OU escalade signal 
  → reconsidérer S26
```

**Situation S25 réelle** :
- Sommeil : 7h11 min (cible atteint > 7h10, **mais < 7h20 limite supérieure**) → **BORDERLINE**
- Énergie : bonne améliorée ✓
- Signaux : **escalade lombaire** → viole "stables" ✓

**Tension logique** :
Deux critères atteints (sommeil proche, énergie OK), mais escalade signal rend décision S26 ambigüe :
- Si l'on applique strictement critère "signaux stables" → **reconsidérer S26** (escalade observée)
- Si l'on applique sommeil atteint + énergie OK → **continue Phase 4 S26** (malgré escalade légère)

**Implication** : Plan S25 reconnaît risque ("si escalade → reconsidérer"), mais escalade modérée (2/4, pas insupportable 4/4) vs escalade sévère rend seuil décision incertain.

- Mardi : courbatures fessiers modérées + douleur lombaire légère (1→0 rapide)

**Impact** :

**Vérifiabilité** : Consult [activities_registry.md](../../../activities/activities_registry.md) ou [FB-30-BASE-V1.md](../../../activities/fullbody/FB-30-BASE-V1.md) spec.

**Priorité clarification** : **MODÉRÉE** — contribue hypothèse cofacteur, mais impact limité si surdosage très modéré.

**Incertitude** :

### Q4 : Durée RUN-40-EF +20 min — intentionnel ou timing accidentel ?

**Incertitude** :
- Plan notation : RUN-40-EF-V1 = 40 min standard
- Exécution : 1h ("pour boucler la boucle, en alternant marche/course sur la fin")
- Écart : intentionnel (choix utilisateur) vs accidentel (timing lost track)

**Impact** :
- Si intentionnel (boucle géographique priorité) → surdosage conscient, risque maîtrisé
- Si accidentel (distraction timing) → signal de fatigue cognitive (attention) possible

**Vérifiabilité** : Utilisateur intention.

**Priorité clarification** : **MINEURE** — intensité RUN-40-EF basse (RPE 4), écart documenté, justification fournie ("boucler la boucle"), impact modéré sur charge. Mais clarification rassurerait sur lucidité charge musculaire.
