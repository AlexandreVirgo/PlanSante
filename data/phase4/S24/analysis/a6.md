## 3. Points d'attention du plan

### Hygiène sommeil (nouveau focus, non-technique)

**Justification** :
- S24 : baisse sommeil -32 min ; utilisateur a identifié multifactorialité (stress professionnel + Phase 4)
- S25 attendu : stress professionnel réduit, Phase 4 continue
- Opportunité : isoler et optimiser facteurs contrôlables (posture, timing repas, environnement)
- Non du ressort d'activités physiques → formalisé en "Consignes particulières" du plan

**Impacts** :
- Repas timing (2–3h avant coucher) → liaison nutrition-sommeil
- Posture bureau (alterner assis-debout) → liaison ergonomie-sommeil (signal lombaire S24)
- Environnement (température, lumière) → facteur externe optimisable

### Monitoring genou (prévention)

**Justification** :
- S24 : signal très léger RUN-45-EF, non-persistant
- Historique : syndrome essuie-glace chronique (plusieurs années)
- Prévention : monitoring RUN-40-EF (durée similaire à RUN-45, intensité plus basse)
- Inclusion instruction S25 : "Si signal genou émerge → réduire intensité ou raccourcir"

**Impacts** :
- Adaptation immédiate pendant séance (utilisateur compétent, démontré S24)
- Pas d'escalade charge → gestion active possible
