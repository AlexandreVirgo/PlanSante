## 2. Tensions structurantes

### Tension 1 : Déficit énergétique intentionnel × Baisse sommeil

**Énoncé** : Phase 4 impose restriction calorique (~300–400 kcal/j via féculents) pour provoquer décrue pondérale. Baisse sommeil (-32 min) suggère accumulation charge métabolique (restriction + activité).

**Manifestations** :
- Sommeil baisse : 7h38 → 7h06 (-32 min)
- Énergie rapportée "moyenne basse" (correspondance directe)
- Motivation reste bonne (non-erosion volitive)

**Cadre logique** :
- S23 (semaine 1 Phase 4) : probablement sommeil stable
- S24 (semaine 2 Phase 4) : baisse émerge
- Hypothèse : accumulation 2 semaines restriction début à se manifester en récupération nocturne

**Résolution structurelle possible** :
- **Option A** : Poursuite Phase 4 + monitoring sommeil (S25, S26)
- **Option B** : Moduler restriction (augmenter féculents légèrement) pour préserver sommeil
- **Option C** : Réduire volume activité pour décompresser charge métabolique
- (Humain décide parmi options)

**Aucune recommandation explicite** (PS-STR-01 ne prescrit pas)

### Tension 2 : Signal lombaire hors-activité × Posture professionnelle

**Énoncé** : Signal lombaire léger mardi-mercredi, uniquement lors station debout prolongée bureau. Changement posture (assis) → soulagement efficace. Aucun signal pendant activités physiques malgré renforcement tronc/hanches.

**Manifestations** :
- Signal : ergonomique, comportemental (posture bureau)
- Gestion : adaptative et efficace (utilisateur)
- Non-persistance : aucun signal en activité physique

**Cadre logique** :
- Signal n'est pas lié à charge d'entraînement (qui respecte design Phase 4)
- Signal est lié à facteur externe non-physiologique (posture bureau)
- Gestion comportementale démontrée efficace

**Résolution structurelle possible** :
- **Option A** : Formaliser rituel posture bureau (changement assis-debout fréquent)
- **Option B** : Maintenir renforcement tronc/hanches FB comme prévention structurelle
- **Option C** : Pas d'action (signal bénin et maîtrisé)
- (Humain décide parmi options)

**Aucune recommandation explicite**

### Tension 3 : Variété nutritionnelle × Cadre restriction

**Énoncé** : Bilan rapporte "léger manque d'inspiration sur variété des légumes". Phase 4 impose restriction féculents (levier principal). Possible : restriction psychologique élargit à légumes (variété basse).

**Manifestations** :
- Variété légumes : "manque d'inspiration"
- Féculents : cadre Phase 4 strict appliqué
- Protéines/lipides : respect

**Cadre logique** :
- Légumes ne sont pas restreints en Phase 4 (volume élevé demandé)
- Manque variété = possible engagement motivationnel baissé (lié énergie basse ?)
- Ou : simple fatigue cognitive de meal planning (pas structurel)

**Résolution structurelle possible** :
- **Option A** : Pré-planner légumes variés (légumes cuits soir, priorité)
- **Option B** : Pas d'action (confort alimentaire secondaire si repos)
- **Option C** : Augmenter variété via épices/assaisonnements (zéro calorique, satisfaction)
- (Humain décide parmi options)

**Aucune recommandation explicite**
