## 2. Options stratégiques

### OPTION 0 : CONTINUITÉ STRICTE (par défaut)

**Définition** :
Poursuivre Phase 4 sans modification jusqu'à S26 (fin de la 4e semaine), en respectant strictement le cadre défini. Décision formelle fin S26 sur passage en stabilisation vs. autre trajectoire.

**Conditions d'activation** :
- Pas de signal critique présent (sommeil > 7h en S25, pas d'escalade signal articulaire)
- Motivation et humeur restent stables
- Digestion tolérée

**Implications** :
- **Semaines** : S25 et S26 en Phase 4 (cadre actuel)
- **Monitoring** : Suivi standard (poids, sommeil, signaux)
- **Décision** : S26 soir = "atteint palier ? → stabilisation" vs. "pas atteint ? → prolonger Phase 4 ou exit"

**Avantages** :
- ✓ Respect design Phase 4 (bornée, réversible)
- ✓ Temps pour observation complète (2 sem remaining)
- ✓ Minimise variabilité décisionnelle
- ✓ Compatible avec "progressivité" (contrainte registre)

**Risques** :
- ⚠ Si sommeil dégringolade S25 (< 7h), poursuite Phase 4 violerait "priorité santé"
- ⚠ Si signal genou/lombaire escalade, charge maintenue pourrait aggraver
- ⚠ Énergie persistant basse → fatigue cumulative S26

**Critères de sortie Option 0** :
- S25 sommeil < 7h00 → escalade Alerte Santé (trigger Décision précoce S25)
- Signal genou ou lombaire escalade vers Niveau 2+ → réévaluation Phase 4
- (Sinon : poursuite S26, décision formelle fin S26)

---

### OPTION 1 : POURSUITE PHASE 4 + MODULATION NUTRITION (féculents +)

**Définition** :
Continuer Phase 4 mais augmenter légèrement apports féculents (approx. +20–30 g/jour) dès S25, dans objectif de stabiliser sommeil sans renoncer à décrue pondérale.

**Conditions d'activation** :
- S24 données suggestent "baisse sommeil × restriction Phase 4" comme tension centrale
- Hypothèse : déficit calorique trop agressif pour récupération nocturne
- Action corrective : modulation féculents = réduction déficit de ~100–150 kcal/jour

**Implications** :
- **Semaines** : S25 + S26 en Phase 4 *modifiée* (féculents relevés)
- **Nutritionnel** : Féculents ~110–150 g/j (vs 80–120 g/j) sur jours mixtes ; rester sans féculent soir
- **Impact pondéral** : Décrue ralentie (peut-être ~0.5 kg/sem au lieu de 1.2 kg/sem)
- **Objectif** : Atteindre ~76 kg (vs 75–75.5 kg) comme palier, acceptable per registre

**Avantages** :
- ✓ Préserve sommeil (par design : moins de déficit)
- ✓ Maintient activité (plus de carburant disponible)
- ✓ Respecte contrainte "priorité santé" de manière préventive
- ✓ Réversible : peut revenir Phase 4 strict post-stabilisation

**Risques** :
- ⚠ Sommeil peut ne pas améliorer (cause externe ?)
- ⚠ Décrue ralentie → palier 76 kg vs 75 kg = plus éloigné de cible long-terme 72 kg
- ⚠ Peut signaler fatigue en restriction, non nécessaire alléger

**Critères de sortie Option 1** :
- S25 sommeil > 7h20 : Option 1 a fonctionné, poursuivre S26
- S25 sommeil 7h00–7h20 : state ambiguous ; décision humaine S25
- S25 sommeil < 7h : escalade alerte ; considérer Option 2 (exit)

---

### OPTION 2 : SORTIE PHASE 4 ANTICIPÉE (fin S24 → stabilisation S25)

**Définition** :
Clôturer Phase 4 à la fin de S24 (demain) et basculer immédiatement en phase de **stabilisation structurée** autour du palier actuel (~75.5 kg). Sortie anticipée justifiée par signaux sommeil/énergie émergents.

**Conditions d'activation** :
- Registre dit "sortie Phase 4 → stabilisation" ; Option 2 accélère timeline
- Justification : "Mieux sortir prématurément une phase d'alerte que d'ignorer signaux santé"
- S24 a atteint ~75.5 kg (très proche cible 75–75.5 kg) → palier déjà présent

**Implications** :
- **Semaines** : S25 + S26 + N en stabilisation (nouvelle phase structure)
- **Nutritionnel** : Féculents remontent à niveau stabilisation (appx. 150–200 g/j)
- **Activités** : Maintien volume (pas escalade) ; focus prévention signaux
- **Objectif** : Stabiliser poids ~75.5 kg, observer sommeil/énergie récupération, valider durabilité avant itération future

**Avantages** :
- ✓ Priorité santé (sommeil) appliquée preactivement
- ✓ Sortie "propre" : palier déjà atteint, pas inachèvement
- ✓ Permet observation long-term durabilité (stabilisation = test "peut-on tenir ce poids ?" sur 4–8 sem)
- ✓ Repos métabolique : organisme se stabilise avant itération future

**Risques** :
- ⚠ Décrue stop abruptement : peut frustrer si momentum "on track"
- ⚠ Palier 75.5 kg vs cible long-term 72 kg = écart 3.5 kg (non negligible)
- ⚠ Peut être perçu comme "sortie de secours" plutôt que décision stratégique

**Critères de sortie Option 2** :
- Immédiate (décision S24 soir) → stabilisation S25
- Post-décision : monitoring stabilisation (sommeil should recover, poids stable)

---

### OPTION 3 : PAUSE + RÉASSESSMENT (fin S24 → bilan semaine courte)

**Définition** :
Interrompre Phase 4 pour une semaine de "réassessment" (S25) en état nutritionnel neutre (retour apports normaux, sans restriction, sans surplus). Objectif : déterminer cause (externe vs. plan) de baisse sommeil.

**Conditions d'activation** :
- Hypothèse : baisse sommeil peut être externe (stress, environnement, charge professionnelle) plutôt que liée Phase 4
- S25 neutre = isoler variable Phase 4 pour cause analysis
- Permet distinguer "est-ce Phase 4 ou coincidence externe ?"

**Implications** :
- **S25 nutrition** : Apports stabilisation (féculents normaux ~150–200 g/j, pas restriction)
- **S25 poids** : Remontée possible (-0 à +0.5 kg eau/glycogène) ; normale et temporaire
- **S25 sommeil** : Observation (devrait améliorer si déficit cause ; devrait rester bas si externe)
- **Décision S25 soir** : Si sommeil améliore → déficit était le coupable → redémarrer Phase 4 ou modulation ; si idem → cause externe → Phase 4 ok

**Avantages** :
- ✓ Diagnostic precis (isolate variable)
- ✓ Pause mentale : interruption perception "restriction continue"
- ✓ Sommeil peut recover complètement
- ✓ Clarité pour décision S26

**Risques** :
- ⚠ S25 "perdue" (pas progession, poids peut remonter)
- ⚠ Perte momentum psychologique (en route → interruption)
- ⚠ Peut être ressenti comme "back to square one"

**Critères de sortie Option 3** :
- S25 soir : si sommeil > 7h20 → Phase 4 était coupable → redémarrer Phase 4 modifiée (Option 1) ou standard (Option 0)
- S25 soir : si sommeil < 7h → cause externe → justifie continuer Phase 4 (Option 0)

## 4. Conditions et points de décision

### Point de décision principal (S24 soir)

**Question clé à l'humain** :
"Quelle posture adopter vis-à-vis de la baisse sommeil et des signaux corporels émergents ?"

1. **Continuer Phase 4 sans changement** (Option 0) → assume signals sont tolérables, observables
2. **Continuer Phase 4 + atténuer déficit** (Option 1) → assume Phase 4 peut être optimisée via modulation
3. **Sortir Phase 4 immédiatement** (Option 2) → assume priority santé > decrue rapide
4. **Pause diagnostic une semaine** (Option 3) → assume need to isolate cause sleep change

### Points de décision secondaires (S25 soir)

**Si Option 0 choisi** :
- Sommeil S25 > 7h10 ? → Continue Option 0 until S26
- Sommeil S25 < 7h00 ? → Escalade alerte ; décision S25 (switch to Option 1 or 2)

**Si Option 1 choisi** :
- Sommeil S25 > 7h20 ? → Continue Option 1 until S26
- Sommeil S25 < 7h ? → Alerte ; switch to Option 2

**Si Option 2 choisi** :
- Stabilisation begin S25 ; monitoring sommeil/poids
- S25 sommeil > 7h20 ? → OK, continue stabilisation
- S25 sommeil < 7h ? → Alerte external cause ; consider separate investigation

**Si Option 3 choisi** :
- S25 sommeil > 7h20 + Phase 4 was culprit ? → Restart Option 1 or 0 S26
- S25 sommeil < 7h + cause external ? → Restart Phase 4 S26 (confidence higher)

## 5. Contraintes non négociables appliquées

Toutes les options respectent registre :
- ✓ **Priorité santé** : Sommeil préservé / monitoé (no option violates < 7h)
- ✓ **Progressivité** : Pas d'escalade simultanée charge + déficit (all options maintain activity volume)
- ✓ **Durabilité** : Pas de restrictions extrêmes (all options use moderated ranges)
- ✓ **Autonomie** : Pas de dépendance outils lourds (all options use simple tracking)
