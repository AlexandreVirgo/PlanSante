# PLAN SEMAINE 28

**Contexte** : Semaine 2/4 Phase 5 (stabilisation structurée). Décision CIH S27 = Option 2 (continuité Phase 5 + observation poids 10–14 jours, sans correction immédiate ; FC en info).

## Objectifs de la semaine

- Priorité lombaire : aucune douleur persistante ; si signal, retour rapide à 0/4.
- Sommeil : préserver la durée ; garder la cible 22h30/23h comme repère la majorité des nuits.
- Poids : observer (pas de sur-correction) ; continuer les pesées comparables.
- Activités : 3 séances (1 renfo + 2 run) avec 1 seule séance qualité strictement contrôlée.

## Alimentation

RAS — appliquer les repères Phase 5 (stabilisation) ; observation 10–14 jours sans micro-ajustement tant qu’il n’y a pas de dérive durable.

## Activités physiques

### Activité 1

**Jour conseillé**: Lundi ou mardi  
**Type**: FB-30-BASE-V1  
**RPE cible**: 5–6  
**Focus particulier**: tronc + hanches (stabilité lombaire), qualité du mouvement  
**Instructions complémentaires**: exécution propre ; arrêter si un signal lombaire apparaît ; ajouter 2 séries de pont fessier si utile.

### Activité 2

**Jour conseillé**: Jeudi  
**Type**: RUN-40-EF-V1  
**RPE cible**: 2–3  
**Focus particulier**: endurance fondamentale stricte, relâchement, “facile” (talk test)  
**Instructions complémentaires**: piloter aux sensations ; rester volontairement facile (conversational) ; si signal lombaire/genou apparaît, réduire immédiatement l’intensité.

### Activité 3

**Jour conseillé**: Samedi  
**Type**: RUN-40-FARTLEK-V1  
**RPE cible**: 2–3 (base) / 6 (accélérations)  
**Focus particulier**: qualité contrôlée, préserver la posture  
**Instructions complémentaires**: privilégier un itinéraire plat/prévisible ; départ conservateur ; si vigilance technique baisse ou si un signal lombaire monte, couper immédiatement les accélérations.

## Consignes particulières

- Lombaire : monitoring quotidien 0–4 ; priorité absolue à l’absence de douleur persistante.
- Sommeil : repère coucher 22h30/23h ; si sommeil ↓, réduire alcool à 0 et dîner plus léger (Phase 5 garde-fous).
- Si ajout d’une séance (optionnel, non planifié) : privilégier un format facile (ex. RUN-30-RECUP-V1), sans cumuler avec la séance qualité ; stop si signal.

## Indicateurs de succès

- Lombaire : RAS ou signal transitoire avec retour rapide à 0/4.
- Sommeil : durée globalement stable ; repère coucher tenu une majorité des nuits (écarts tolérés si non déstabilisants).
- Poids : pas de dérive durable sur 10–14 jours.
- Activités : 3/3 réalisées sans escalade de signaux ; séance qualité “contrôlée”, pas subie.
