# GRILLE DE RÉFÉRENCE RPE (1–10)
## Borg Scale appliquée aux activités FB & RUN

**Objet** : Calibration commune du RPE pour auto-évaluation cohérente lors de l'exécution des activités.  
**Public** : Néophyte progressant, utilisation sans équipement (FC optionnelle).  
**Révision** : à chaque phase ou adaptation perçue.

---

## 0. Comprendre le RPE : définition courte

| Concept | Signification |
|---------|---------------|
| **Ressenti** | Comment le corps *se sent*, pas la performance |
| **Échelle 1–10** | 1 = repos ; 10 = épuisement maximal |
| **Context-spécifique** | RPE 5 en running ≠ RPE 5 en FB (durée, type d'effort) |
| **Signal du jour** | Intègre sommeil, digestion, stress, fatigue chronique |

---

## 1. Grille universelle (1–10) pour auto-évaluation

### RPE 1–2 : Très facile (RÉCUPÉRATION)
**Ressenti physique** :
- Respiration très contrôlée, parler sans essoufflement
- Pas de sensation de fatigue
- Chaleur corporelle minime
- Mouvements fluides, aucune tension

**Running** :
- Allure conversationnelle aisée
- Jambes légères
- FC estimée : < 120 bpm

**Full Body** :
- Échauffement, mobilité, récupération légère
- Aucun essoufflement
- Exercices faciles (étirements, mouvements lents)

**Exemples réels** : RUN-30-RECUP-V1 ; FB-30-MOBI-RENFO-V1 (base)

---

### RPE 3–4 : Léger-modéré (BASE AÉROBIE)
**Ressenti physique** :
- Respiration montante mais contrôlée
- Peut parler en phrases entières
- Légère chaleur corporelle
- Jambes actives mais pas lourdes
- Sensation de "travailler sans forcer"

**Running** :
- Allure soutenue mais confortable
- Pace constant
- FC estimée : 120–145 bpm
- Pourrait continuer longtemps

**Full Body** :
- Légère accélération du cœur lors de circuits
- Respiration élevée mais contrôlée
- Exercices simples avec tempo régulier

**Exemples réels** : RUN-40-EF-V1 ; RUN-60-SL-V1 (base)

---

### RPE 5–6 : MODÉRÉ (EFFORT SOUTENU)
**Ressenti physique** :
- Respiration manifeste, phrase courte possible
- Chaleur corporelle nette (peut retirer gilet/manche)
- Jambes "actives" mais rythmée
- Impression de "travailler significativement"
- Sueur légère à modérée

**Running** :
- Allure claire et soutenue
- Effort constant, pas de pause
- FC estimée : 145–160 bpm (endurance active)
- Pourrait tenir 30–45 min

**Full Body** :
- Circuit continu, respiration rapide
- Sueur visible
- Exercices à cadence régulière, burn musculaire léger

**Exemples réels** : FB-30-BASE-V1 ; RUN-40-FARTLEK-V1 (blocs base)

---

### RPE 7–8 : ÉLEVÉ (SEUIL / INTENSITÉ)
**Ressenti physique** :
- Respiration rapide, parler par mots isolés
- Sueur abondante
- Jambes lourdes mais contrôlées
- Sensation d'effort "sérieux"
- Accumulation lactique (jambes, cœur)

**Running** :
- Allure proche du seuil anaérobie
- Effort soutenu et concentré
- FC estimée : 160–175 bpm
- Tenir 20–30 min max (effort concentré)

**Full Body** :
- Densité maximale, circuit rapide
- Respiration très rapide
- Exercices explosifs ou lourds
- Muscles "brûlent" légèrement

**Exemples réels** : FB-30-DENSITÉ-V1 ; RUN-40-TEMPO-FRACTIONNE-V1 ; RUN-45-SEUIL-V1 (segments clés)

---

### RPE 9–10 : MAXIMAL (ÉPUISEMENT)
**Ressenti physique** :
- Respiration maximale, incapacité à parler
- Sueur extrême
- Jambes lourdes, coordination difficile
- Fatigue musculaire intense
- Sensation "je n'en peux plus"

**Running** :
- Sprint, allure maximale
- FC estimée : > 175 bpm
- Tenir < 10 min

**Full Body** :
- Exercices extrêmes, explosive
- Récupération nécessaire après

**Usage PlanSanté** : **À éviter volontairement** — cet effort n'est jamais prescrit.  
→ Si RPE 9–10 observé, c'est **surcharge ou mauvaise gestion** à analyser.

## 2. Calibration progressive (pour néophyte)

### Phase de découverte (Semaines 1–4)
**Objectif** : apprendre à reconnaître les seuils RPE personnels.

**Tâche à chaque séance** :
1. Notation du RPE ressenti **à chaud** (fin de séance ou au milieu)
2. Note brève du contexte : sommeil, digestion, moral, météo
3. Observation FC si disponible → mettre en lien avec ressenti

**Outil** : Log simple dans chaque `bilan_semaine_XX.md`  
Format :
```
### Calibration RPE (découverte)
| Activité | Date | RPE déclaré | Contexte | FC (si mesurée) | Observations |
|----------|------|-------------|----------|-----------------|--------------|
| RUN-40-EF-V1 | J lun | 3 | Bien dormi, météo facile | 135 bpm | Respiration contrôlée, pas d'essoufflement |
| FB-30-BASE-V1 | J jeu | 5 | Digestion lourde | 145 bpm (pic) | Jambes lourdes, effort plus dur |
```

### Après calibration (Semaine 5+)
- Utiliser le RPE comme **signal primaire** d'ajustement
- Croiser avec FC seulement en cas de **doute** ou **anomalie**

## 3. Pièges courants à éviter (néophyte)

| Piège | Risque | Solution |
|-------|--------|----------|
| **Confondre FC et RPE** | "J'ai 140 bpm donc RPE 5" (faux) | RPE = ressenti global, FC est une donnée secondaire |
| **Dérive dans le temps** | RPE augmente imperceptiblement (adaptation) | Recalibrer tous les 2 mois en confrontant avec ressenti honnête |
| **Ignorer le contexte** | Même RPE déclaré mais conditions variables | Noter systématiquement : sommeil, digestion, météo, stress |
| **Forcer la conformité** | "Je dois faire RPE 3, mais je sens RPE 5" | **Le ressenti prime toujours** ; documenter la déviation |
| **Variation jour/jour excessive** | RPE 3 lundi, RPE 6 jeudi même activité | Signal possible de fatigue, stress ou mauvais sommeil → action pour jour suivant |

## 4. Application pratique : exemples de lectures RPE

### Exemple 1 : RUN-40-EF-V1 (40 min, cible RPE 2–3)
**Pendant la séance** :
- ✓ Min 10 : respiration montante mais contrôlée → **RPE 2–3**
- ✓ Min 25 : jambes fluides, peut parler → **RPE 3 confirmé**
- ⚠ Min 35 : jambes commencent à peser → **RPE monte à 4** (normal, dernier 1/4)
- ✓ Fin : léger essoufflement, pas de fatigue → **RPE final 3**
→ **Séance conforme** (objectif 2–3 atteint)

---

### Exemple 2 : FB-30-BASE-V1 (3 tours, cible RPE 5–6)
**Tour 1** :
- Repos, peu d'essoufflement → **RPE 2**

**Tour 2** :
- Respiration montante, chaleur notable → **RPE 4–5**

**Tour 3** :
- Sueur visible, respiration rapide, jambes actives → **RPE 5–6**
- Fin de tour 3 : pic d'effort → **RPE 6**

→ **Séance conforme** (objectif 5–6 atteint à la fin)

---

### Exemple 3 : FB-30-DENSITÉ-V1 (circuit continu, cible RPE 6–7)
**Min 10** : chaleur corporelle nette, respiration rapide → **RPE 6**  
**Min 20** : sueur abondante, jambes lourdes, parler difficile → **RPE 7**  
**Fin** : respiration très rapide, fatigue musculaire → **RPE 7–8**

→ **Séance conforme** (objectif 6–7 atteint)

## 5. Synthèse : tableau de navigation rapide

| RPE | Sensation clé | Running | Full Body | Usage |
|-----|---------------|---------|-----------|----|
| 1–2 | Très facile, pas de fatigue | RUN-30-RECUP | FB-30-MOBI | Récupération |
| 3–4 | Léger, contrôlé, conversationnel | RUN-40-EF, RUN-60-SL | FB-30-MOBI (+ travail) | Base aérobie |
| 5–6 | Soutenu, respiration claire | RUN-40-FARTLEK (blocs) | FB-30-BASE, FB-30-DENSITÉ | Effort structuré |
| 7–8 | Élevé, parler difficile, sueur | RUN-45-SEUIL, RUN-40-TEMPO | FB-30-DENSITÉ (fin), FB-40-FORCE | Intensité |
| 9–10 | Épuisement maximal | Sprint court | Rarissime | **À éviter volontairement** |

**Responsable** : PS-PLAN-08 (sélection), utilisateur (calibration personnelle)

**Entrées** : Observations de séances, contexte, FC (optionnelle)
